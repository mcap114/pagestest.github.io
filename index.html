Turn-based Tile Shooter

Created a simplified turn-based, tile-based shooter game in C. The game features a grid where players and enemies move, allowing players to move horizontally and shoot enemies in the same column. Enemies follow a predictable movement pattern, and the game includes scoring as well as win/loss conditions based on player performance and enemy progression.

Shopping App

Built a shopping application in C that offers user registration, item listing, inventory management, and shopping cart capabilities. Users can purchase, sell, and edit stock while viewing products by category or search. Admin functionalities are included for user management and sales oversight.
